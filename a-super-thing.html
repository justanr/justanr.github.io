Windowing With Accumulate

  • Needs initial value...
  • Needs to maintain state...
  • An object, maybe? A closure?
  • Closures are cooler...
from itertools import accumulate

def shift_group_left(group, val):
    return tuple(group[1:]) + (val,)

assert shift_group_left((4,5), 6) == (5,6)

Accumulate's callable needs to accept two values:

  • The current state
  • The next value
def shifter(initial):
    def _shifter(cur, val):
        nonlocal initial
        initial = shift_group_left(initial, val)
        return initial
    return _shifter

list(accumulate("mary had a little lamb".split(), func=shifter((None, None))))

['mary', (None, 'had'), ('had', 'a'), ('a', 'little'), ('little', 'lamb')]
